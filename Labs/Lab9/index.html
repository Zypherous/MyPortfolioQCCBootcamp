Internal CSS

Internal CSS is css for controlling a specific page from within the page. If you have multiple pages all using the same style sheet, but have only one page you want to be different, you can use internal css with the "<style></style>" tag and then using css syntax to write the rule-set for the page. It should be above all elements that are to be styled.

External CSS

External CSS is css styling that lives in another file. It's extension is .css and uses the following syntax: "selector{ property: value;}" It can be linked in the html file with the <Link href="path to css" rel="stylesheet" type="text/css"/>
